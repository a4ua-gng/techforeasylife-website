# Tech for Easy Life website

Responsive school-facing website for Tech for Easy Life (TEL), a student-led STEM and model-rocketry initiative based in Guwahati, Assam.

## Preview locally

Open `index.html` in a browser, or serve this folder through any static file server.

## Before production launch

- Replace the temporary email and WhatsApp links with TEL's official contacts.
- Add TEL's approved logo, rocket photography and launch video.
- Confirm final institutional pricing and wording.
- Complete accessibility and cross-browser checks.
