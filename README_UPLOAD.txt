HOW TO UPLOAD
1. Extract this ZIP.
2. Open your GitHub repository.
3. Upload all files from this folder into the repository root.
4. Replace the existing files when GitHub asks.
5. Keep your existing image files, styles.css and the other pages/assets already in the repo.
6. Commit the upload and wait for GitHub Pages to redeploy.
7. Hard-refresh the site after deployment (Ctrl+Shift+R on Windows).
